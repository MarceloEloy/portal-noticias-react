# Portal de Notícias 📰

Projeto front-end desenvolvido como atividade acadêmica da disciplina **Front-End Avançado** do curso de **Ciência da Computação**.

O projeto consiste em um **Portal de Notícias** com páginas de Home, Contato e Login. Na Home, é possível visualizar notícias, buscar por palavras-chave, curtir notícias, abrir o conteúdo completo e criar comentários. Todo o sistema é construído apenas no front-end, com armazenamento local utilizando **LocalStorage**, e estilizado com CSS personalizado. Além disso, foram implementados testes automatizados com **Vitest**, que podem ser visualizados via **Vitest UI**.

## ⚙️ Funcionalidades

- Páginas **Home**, **Contato** e **Login**.
- Exibição de notícias com conteúdo detalhado.
- **Busca de notícias** através da search bar.
- **Curtir notícias**.
- **Comentários** nas notícias.
- Armazenamento de dados no **LocalStorage**.
- **Testes automatizados** com Vitest e interface do Vitest UI.
- Estilo totalmente personalizado com **CSS**.

## 🧠 Aprendizados

- Construção de um front-end completo com **React e Vite**.
- Manipulação do **LocalStorage** para persistência de dados.
- Implementação de **componentes reutilizáveis**.
- Gestão de estado e props no React.
- Criação e execução de **testes automatizados** com Vitest.
- Estruturação e organização de um projeto front-end escalável.
